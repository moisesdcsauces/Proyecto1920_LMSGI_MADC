Lenguaje de Marcas y Sistemas de Gestion de Informacion:


El módulo de Lenguajes de Marca y Sistemas de Gestión de Información se encuentra dentro de los tres ciclos formativos del catálogo público de Formación Profesional: DAW, DAM Y ASIR.
Es un modulo que implementa diferentes tipos de lenguajes(html, css, xml, xpath, xslt), todos ellos relacionados con páginas web y consultas a éstas.
